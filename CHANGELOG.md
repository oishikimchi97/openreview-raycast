# OpenReview Changelog

## [Initial Version] - {PR_MERGE_DATE}

- List your OpenReview submissions across all venues, newest first (by true
  submission time).
- Per-venue review scores with the venue's own label (Rating / Overall
  Recommendation / Overall Assessment …) and average / min / max.
- Status detection: Under Review, Reviewed, Accepted, Rejected, Withdrawn,
  Desk Rejected — with active-phase (open rebuttal/review window) detection so a
  paper still in rebuttal reads as Under Review rather than Reviewed.
- Status filter dropdown.
- Detail page: abstract, colored-tag metadata, per-reviewer scores with links to
  each review.
- 💡 new-review indicator (tracked locally) with mark-as-seen.
